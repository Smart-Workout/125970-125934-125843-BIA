# Smart Workout DSS/BIS Project

This repository contains the project proposal, datasets, reference material, and planning notes for the Smart Workout DSS/BIS project.

## Folder Structure

- `proposal/` - current project proposal source, PDF, and outline notes.
- `data/raw/` - original CSV datasets used for the project.
- `notebooks/` - exploratory notebooks and analysis work.
- `requirements/` - professor requirements and assignment brief.
- `references/` - supporting documents, examples, and report templates.
- `TODO.md` - working task list and project planning notes.

## Git Workflow

After making a new change, check what changed:

```powershell
git status
```

Then commit the new addition separately:

```powershell
git add .
git commit -m "Organize project folder structure"
```
